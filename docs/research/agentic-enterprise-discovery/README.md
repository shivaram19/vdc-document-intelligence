# Agentic Enterprise Discovery — Question Index

**Purpose:** A structured set of research questions to discover, in the open-source and human-centered world, what it actually takes to build an agentic enterprise. Each file focuses on one domain so we can answer questions deeply rather than dump everything in one place.

---

## How to Use These Files

1. Pick a question file based on what decision is blocked.
2. Answer the sub-questions with evidence, not opinion.
3. Update the "Current Hypotheses" and "What Would Change Our Mind" sections as we learn.
4. Add citations in `docs/citations/AGENTIC_ENTERPRISE_DISCOVERY_001.bib`.
