# Research & Citation Registry

## Rule
Every line of code, every architectural choice, every optimization must be traceable to:
1. A **peer-reviewed paper** (preferred)
2. An **industry standard / spec** (ISO, NIST, etc.)
3. A **primary source** (official docs, RFC, source code)
4. A **documented trade-off decision** (ADR in `docs/decisions/`)

## Citation Format (in code)
```python
# [CITE: Shahidinejad2021] Short-TTL capability tokens reduce attack surface
# [CITE: AIP2026] Agent Identity Protocol — delegated token exchange
# [CITE: RedHat2026] Zero Trust for agentic AI — continuous verification
```

## Citation Format (in ADRs)
```markdown
## References
- [Shahidinejad2021] Shahidinejad et al. (2021). "A Secure Token-based..." [URL]
- [AIP2026] Agent Identity Protocol (2026). Red Hat / IBM Research. [URL]
```
