# Discord

**Servers:** 8020BIM Community, Revit API Discord (discord.gg/WMXVS6MT6C)

## Pain Signals
- 8020BIM: Paid community proves people pay for BIM knowledge
- Revit API Discord: Developers automating documentation workflows
- Automation is #1 topic = early adopter audience

## Tactics

### Be the Helpful Expert (30-day rule)
Join 8020BIM. Don't pitch for 30 days.  
Answer questions about document management, specs, RFIs.  
Build reputation as "the document guy/gal".  
Then: *"By the way, I built a tool that does this automatically..."*

### Revit API Discord — Technical Angle
Post: *"We built an agentic AI system that reads construction docs and exposes tools via MCP. Looking for beta testers who understand both BIM and AI."*

### Create Your Own Server
- Name: "Construction Document Intelligence"
- Invite BIM professionals from LinkedIn
- Weekly "Contradiction of the Week" — real spec-drawing conflicts

- Helpful answers: 5/week
