# India-Specific Platforms

## Where Indian Construction Professionals Live

| Platform | Audience | Strategy |
|----------|----------|----------|
| LinkedIn India | BIM managers, VDC leads | Hindi + English content |
| IndiaMART | SMB contractors | List as service |
| Sulekha / JustDial | Local contractors | Local SEO + WhatsApp |
| WhatsApp Groups | Project teams, vendors | QR code demos |
| ACE India / TechnoStruct | BIM students | Partnership |
| Nagpur Metro / DMRC | Public infra | Pilot projects |
| Autodesk India Partner Network | Software resellers | Co-sell |

## India-Specific Pain
- <20% of firms <500 employees have standardized BIM
- Autodesk launched "BIM Package for Viksit Bharat" (June 2025)
- Adani + Autodesk 3-year partnership
- Nagpur Metro 5D BIM = showcase project

## Tactics

### BIM Training Institute Partnerships
*"Your graduates learn BIM modeling. We teach them document intelligence. Joint certification?"*

### WhatsApp Business API
QR code on construction site: *"Ask your project documents a question"*

### Local Language Content
Hindi video: *"10-minute mein spec kaise dhundein"* (How to find specs in 10 minutes)

## Metrics
- WhatsApp QR scans: track monthly
- Training partnership signups: 2 in 90 days
