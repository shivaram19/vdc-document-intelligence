🧠 Core AI Capabilities

✓ Multi-Document RAG

Upload an entire project document set and ask natural language questions. Every answer is grounded in your documents with cited sources and similarity scores.

✓ Auto-Draft RFI Responses

Paste an RFI question; Medha scans all project docs and drafts a professional TO/FROM/DATE/RE response with full citations — in 30 seconds, not 4 hours.

✓ Drawing-Spec Contradiction Detection

AI scans drawings and specs to flag dimension mismatches, conflicting requirements, and code discrepancies before they reach the field.

✓ Query-Aware Scanning

Uses semantic embedding similarity to only flag contradictions relevant to the user's question — no false positive noise. A 0.35 confidence threshold prevents hallucination.

📄 Document & Data Engine

✓ Multi-Format Ingestion

PDF, DOCX, TXT, MD. pdfplumber + python-docx with OCR fallback via pytesseract for scanned PDFs. Optional IBM Docling for layout-aware extraction.

✓ Smart Chunking & Embeddings

512-character chunks with 128-character overlap. all-mpnet-base-v2 (768-dim) embeddings with cosine similarity search. Stored per-project for clean isolation.

✓ Confidence Thresholding

Queries scoring below 0.35 similarity get an explicit "low confidence" message instead of a fabricated answer. No hallucination. No guesswork.

✓ Per-Project Isolation

Each project has its own document index, embedding store, and metadata. Clean separation by client. No data bleed between projects.

How It Works

From Document Dump to Intelligence in Four Steps

Step 01

Create a Project & Upload Documents

Drop PDFs, DOCX, TXT, or MD files into the inbox. Select document types: drawing, spec, RFI, submittal, or addendum. The Inbox Watcher auto-triggers ingestion — no manual processing needed.

Step 02

Parse, Chunk & Embed

Documents are parsed (pdfplumber / Docling), split into 512-char overlapping chunks, and embedded with all-mpnet-base-v2. Vectors are stored per-project with clean client isolation.

Step 03

Query with Natural Language

Ask anything: "What is the concrete strength for columns?" The retriever finds the most relevant chunks via cosine similarity, and the LLM synthesizes a cited answer — grounded, not guessed.

Step 04

Detect Contradictions & Draft RFIs

The contradiction engine scans retrieved chunks for numeric mismatches (psi, psf, ft, in, °F, gpm, cfm). The RFI drafter auto-generates professional responses with full citations — ready to send.
